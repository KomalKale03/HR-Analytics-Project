# HR Analytics Dashboard Project

## Overview
This project demonstrates end-to-end HR Analytics using SQL, Python, and Power BI on a dataset of 2500+ employees. The goal is to analyze attrition trends, department performance, salary distribution, and employee satisfaction for data-driven HR decisions.

## Dataset
- File: `dataset/HR_Analytics_Dataset_2500_Records.csv`
- Columns include EmployeeID, Age, Gender, Department, JobRole, EducationLevel, MaritalStatus, Location, MonthlyIncome, YearsAtCompany, PerformanceRating, Attrition, and more.

## Features
- **SQL Queries:** Joins, aggregations, and KPIs
- **Python:** Data cleaning, EDA using Pandas & NumPy
- **Visualization:** Matplotlib and Seaborn charts
- **Power BI Dashboard:** Three interactive pages:
  1. **Attrition Analysis:** KPI cards, Donut Chart, Attrition by Department & Gender
  2. **Department KPIs:** Salary, Performance, Job Satisfaction, Employee Count
  3. **Performance Insights:** Training vs Performance, Overtime vs Attrition, Top 10 Highest Paid

## Key KPIs
- Total Employees
- Attrition Rate (%)
- Average Salary
- Employee Distribution by Department, Location, Gender
- Performance and Satisfaction Metrics

## Usage
- Open `HR_Analytics_Dashboard.pbix` in Power BI Desktop
- Use slicers for dynamic filtering by Department, Gender, Location, and Marital Status
- Explore the interactive charts for HR insights
